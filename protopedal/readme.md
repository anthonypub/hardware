Idea: A helper PCB for pedal construction, similar to many that have been done.

Will include:

- Power from regular pedal wart + 9V, with auto-switch
- Optional Polarity protection?
- Option for reverse polarity (for the weird transistors)
- Input and output jacks
- 3PDT switching
- LED for power
- LED for effect
- Common Pots? Would be nice to have pre-mounted pots in common sizes to avoid having to solder
    Protis mini includes 1k, 10k, 100k, 1M. I've seen 500Ks to so add those?
- A few switches as well, SPST and SPDT.
    
- Output pins for:
    - Power + Gnd (multiple, more for ground)
    - Direct access to all jack pins
    - Jack pins through switch
    - Optional half-voltage rail


Some resources:
https://learn.sparkfun.com/tutorials/proto-pedal-assembly-and-theory-guide/all (Includes pointers to reverse polarity safety)

Optional: 5v in case I want to do microcontroller stuff.

Very optional: Protis guys have a cool take that plugs directly into a normal breadboard.
